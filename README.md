# Eye Controlled Mouse

### Patent-backed Assistive Technology | Accessibility-focused Solution

## Overview

Eye Controlled Mouse is a real-time assistive technology project designed to enable completely hands-free computer interaction. The system uses eye movements and blink detection to control cursor movement and mouse clicks without requiring a physical mouse or keyboard.

Built using computer vision and facial landmark detection techniques, the solution aims to improve accessibility for individuals with physical or motor disabilities.

---

## Key Features

✔ Real-time eye tracking using MediaPipe facial landmarks  
✔ Gaze-based cursor movement  
✔ Blink detection for mouse clicks  
✔ Works with a standard webcam  
✔ No additional hardware required  
✔ Lightweight and easy to deploy

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Eye_controlled_mouse.git
cd Eye_controlled_mouse
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
